# PrivacyBlur

[English version](./README.md)

PrivacyBlur は、画像内の個人情報や見せたくない部分をブラウザ上でローカルにぼかすための小さなツールです。

## 特徴

- 画像ファイルの読み込みと、クリップボードからのペーストに対応しています。
- 画像上をクリックすると、その位置を中心にぼかしを追加できます。
- ぼかし領域の大きさ、ぼかし強度、境界のなじませを調整できます。
- 直前の操作を取り消し、ぼかしの全消去、PNG/JPEG 保存ができます。
- 画像はブラウザ内で処理し、外部サーバーへ送信しません。

## 使い方

1. ブラウザで [`index.html`](./index.html) を開きます。
2. ファイル選択から画像を読み込むか、`Ctrl+V` / `Cmd+V` で画像を貼り付けます。
3. 左側のパネルでぼかし設定を調整します。
4. 隠したい箇所を画像上でクリックします。
5. PNG または JPEG で保存します。

## プライバシー

このツールはローカル処理を前提にしており、画像データはブラウザ内に留まり、外部サーバーへ送信されません。
