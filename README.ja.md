# PrivacyBlur

画像の機密部分を素早くぼかすための、シンプルでブラウザベースのツールです。すべての処理はブラウザ内でローカルに行われるため、画像がサーバーにアップロードされることはありません。

## 機能

- **どこからでも読み込み:** ローカルファイルから画像を開くか、クリップボードから直接貼り付けることができます（`Ctrl+V` / `Cmd+V`）。
- **インタラクティブなぼかし:** 画像をクリックするだけで、その位置にぼかしマスクを追加できます。
- **細かな調整:** ぼかし領域のサイズ、ぼかしの強さ、エッジのフェザリング（境界のぼかし）を調整して、自然になじませることができます。
- **クライアントサイド設計:** 画像が外部サーバーに送信されることはなく、プライバシーが保証されます。
- **複数のフォーマット:** 最終的な画像をPNGまたはJPEGとしてエクスポートできます。
- **簡単な編集:** ワンクリックで最後のぼかしを取り消したり、すべてのぼかしをクリアしたりできます。

## 使い方

1. Webブラウザで [`index.html`](./index.html) を開きます。
2. ファイル選択を使用するか、クリップボードから貼り付けて画像を読み込みます。
3. 左側のパネルを使用して、ぼかしの設定を調整します:
   - **サイズ:** 画像の幅に対する割合（%）でぼかし領域を調整します（2%〜50%）。
   - **ぼかしの強さ:** ぼかしの半径をピクセル単位で設定します（2px〜40px）。
   - **エッジのフェザリング:** ぼかしマスクの境界を柔らかくし、より自然な見た目にします（0px〜80px）。
4. 画像をクリックしてぼかしマスクを適用します。必要な数だけ追加できます。
5. 「Save as PNG」または「Save as JPEG」をクリックして結果をダウンロードします。

## プライバシー

このツールは、プライバシーを基本原則として設計されています。すべての画像処理は完全にブラウザ内で行われます。ファイルが外部サーバーにアップロード、保存、または送信されることはありません。

## 要件

Canvas APIをサポートするモダンなWebブラウザ（例: Chrome、Firefox、Safari、Edge）。
